Бот позволяющий наладить коммуникации студента и наставника, с наличием IT-форума для образовательной организации. В совменнои мире керайне необходимо цифролизация образоватнльного процесса. У студентов в ходе обучения возникают вопросы которые мешают спать, данный бот позволяет онлайн получить ответ. У отвечающих в качестве дополнительного стимула, есть возможность заработка.
